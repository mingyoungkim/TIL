# 데이터 크롤링의 세계

## 인터넷에서 데이터를 수집하는 방법

1. OpenAPI 등 공개된 API를 사용 ([data.go.kr](http://data.go.kr), naver developer, daum developer 사이트 등)
2. HTTP Get Method
   - 정보가 게시되어 있는 대상 웹사이트를 HTTP Get을 사용하여 html 코드를 얻고 Text Parsing 해서 사용
   - Java / Go / Python / C / C++ / Perl / Node.js 등 거의 대부분의 언어로 구현 가능
3. Selenium Web Driver
   - 웹브라우저 인스턴스를 생성해 실행시킨 후 해당 인스턴스를 컨트롤
     - `브라우저를 열고 네이버로 가자` 커맨드를 날리면 실제 브라우저 인스턴스가 화면에 뜬다. 그 브라우저를 컨트롤해서 특정 사이트로 이동하거나, 특정 데이터를 가지고 오는 것이 가능하다.
     - 사람이 하는 것과 다를 것이 없어서 거의 모든 데이터를 가져올 수 있다.
   - 웹사이트 테스트 자동화 목적으로 개발
   - 가상의 브라우저를 실행시키는 Headless Mode 등이 있음

### 정리

- OpenAPI 등 공개된 API를 사용
- HTTP Get Method를 사용해 HTML을 가져오는 방법
- Selenium Web Driver를 사용하는 방법
- 사람이 직접 수작업



## 웹 크롤러 vs 웹 스크래퍼

- 웹 크롤러 (Web Crawler)
  - 조직적, 자동화 된 방법으로 웹을 탐색/수집하는 프로그램
    - ex) 구글, 네이버 등의 검색엔진 결과 데이터를 수집하기 위한 봇(Bot)
- 웹 스크래퍼 (Web Scapper)
  - 웹 사이트에서 정보를 추출하는 프로그램
    - ex) 상품별 가격을 알기 위해 해당 상품을 파는 페이지들의 가격을 추출
- 크롤러보다는 대부분 단순 스크래퍼 개발 수요가 많음
- 우리나라에서는 많은 기업들이 같은 의미로 혼용



## 웹 크롤링은 불법인가요?

- 웹사이트의 홈디렉토리에 위치한 robots.txt 파일을 열어보고 해당 사이트의 정책을 준수하지 않는다면 불법

- 크롤링한 자료를 상업적인 용도로 사용하면 불법

  - 사람인 vs 잡코리아

    - 사람인에서 경쟁사인 잡코리아 홈페이지에서 채용정보를 크롤링하고 상업적으로 활용했던 사례, 불법으로 판별

  - 여기어때 vs 야놀자

    - 여기어때에서 야놀자의 정보를 활용했던 것도 불법으로 판별

  - HiQ Labs vs Linkedin

    - 로그인 필요없는 정보는 공공재. 크롤링 불법 아니다

      - 내용 : HiQ Labs에서 Linkedin에 올라와있는 사람들의 프로필을 수집하고, 이를 다른 정보들과 함께 가공 / 분석하여 판매

      - 판결: 

        ```
        퍼블릭 정보 크롤링은 불법이 아니다
        ```

        1. 특정 사용자에게 오픈된 (로그인이 필요한) 데이터들이 아닌 것은 누구나 볼 수 있는 퍼블릭 정보이다.
        2. 퍼블릭 데이터는 승인 없이 가져갈 수 있다. 게시자가 가져가지 말라고 해도!

- 비상업적인 용도라 하더라도 원작자에게 불이익을 주면 불법

- 크롤러를 활용해 고의적으로 Abusing

  - 네이버 검색순위 조작
  - 음원 순위 조작
