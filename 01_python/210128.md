> 원 면적 : 반지름^2 * 3.14

> 원 둘레 : 반지름 x 2 x 3.14

```python
# 원 면적 함수
def circle(r):
    return 3.14 *r*r

def circle2(r):
    return r * 3.14 * 2

a = circle(int(input()))
b = circle2(int(input()))
```

> 반목문을 이용해 단어의 순서를 거꾸로 해 반환하는 함수 (회문 판별 함수)

```python
a = input()

def revert(string):
    result = ''
    
    # 끝에서부터 0까지
    for i in range(len(string)-1, -1, -1):
        result += string[i]
    return result

string_input = revert(a)

# 거꾸로 반환한 문자가 input한 문자와 같으면 회문
if a == string_input:
    print(string_input)
    print('입력하신 단어는 회문(Palindrome)입니다.')
else:
    print('입력하신 단어는 회문(Palindrome)이 아닙니다.')
```

> 소수를 검사하는 함수 (사용자가 입력한 숫자가 소수인지를 판단하는 프로그램)

```python
a = int(input())

def inspect(number):
    # 소수 갯수 넣을 변수 초기화
    result = 0    
    
    for i in range(1, number+1):
        num = number % i
        if num == 0:
            result += 1
        else:
            result = result
    return result


number_input = inspect(a)
```

> 피보나치 수열의 결과를 생성하는 프로그램

```python
a = int(input())

def fibonacci(number):
    
    # 1로 된 list를 number수 만큼 만들기
    fibo = [1] * number
    for i in range(2, number):
        
        # 2부터 number까지 범위의
        # i를 2뺀 수와 1뺀 수를 더해서
        # 다시 fibo라는 list에 넣기
        fibo[i] = fibo[i-2] + fibo[i-1]
    return fibo

number_input = fibonacci(a)
print(number_input)
```
