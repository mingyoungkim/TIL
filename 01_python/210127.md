### 반복문 while

> '*' 피라미드 쌓기

```python
i, k = 5, 1
# i는 공백에 대한 변수, k는 *에 대한 변수

while i >= 0:
    print('{}{}' .format(' ' * i, '*' * (2* k -1)))
    
    # 공백은 1씩 줄고
    i = i - 1
    # 별은 늘이자 >_<
    k = k + 1
```




> 거꾸로도 쌓아보자!

```python
# 변수 의미는 위와 같음
i = 0
k = 7
while 0 < k <= 7:
    if k % 2:
        print('{}{}' .format(i* ' ', k * '*'))
        
        # 하지만, 위의 경우와 다르게
        # 공백은 늘고

    k -= 1
```
