### 문자열 처리

> 내장함수 사용해보자

```python
password = 'a124di'

flag_alpha = False
flag_num = False

for i in password:

    # i가 알파벳이면
    if i.isalpha():
        flag_alpha = True

    # i가 숫자면
    if i.isdigit():
        flag_num = True

# flag_alpha 가 True로 바뀌지 않아
# False 라면
if not flag_alpha:
    print('비밀번호에 알파벳이 없음')
    
# flag_num 이 True로 바뀌지 않아
# False 라면
elif not flag_num:
    print('비밀번호에 숫자가 사용되지 않았음')
else:
    print('완벽한 비밀번호')
```

> 문자열 숫자를 정수로 변환하기

`int(), str() 등 내장함수를 사용하지 말고 변환해보자`

- a to i

  ```python
  def atoi(num_str):
      # 최종 값을 담을 변수
      value = 0
      
      # num_str 로 해도 되지만
      # 인덱스 접근을 위해 range(len())해보자
      for i in range(len(num_str)):
          value *= 10
          
          # ord()는 아스키코드 값으로 변환
          # 0을 가지고 와서 아스키코드 값으로 변환해서 빼줌
          value += ord(num_str[i]) - ord('0')
  
      return value
  
  num_str = "1234"
  
  num_int = atoi(num_str)
  print(num_int)
  ```
